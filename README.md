# Projeto de Dispositivo Móvel 
## MobileAppStore - O app permite ao usuário navegar por categorias, ver produtos e comprar de forma prática, mostrando na prática essas vantagens.

### Nessa primeira tela foram criados botões de interação. Botão cadastrar que como pode ser visto, direciona para um Alert.

### Essa imagem mostra a tela inicial de login com campos para digitar email e senha e logo abaixo o botão de Entrar que ao clicar nesse botão o usuário é direcionado para a segunda tela.

<p align="center"><img src="https://github.com/user-attachments/assets/7994421c-2aa1-480c-835b-fb616812d6d5" width="300" height="500"></p>

### Essa é a tela de lista de produtos que está disponibilizada na API utilizada para esse projeto. A lista contém uma lista de produtos dividada em Tabs (categorias masculino e feminino localizados no fundo da tela) e em subcategorias dividindo a lista em tipos de produtos. Para navegar entre as subcategorias basta clicar que o botão selecionado irá se destacar e a lista de produtos dessa categoria irá aparecer logo abaixo. Para verificar detalhes e efetuar a compra deve-se clicar no botão Detalhes para que o usuário seja direcionado para a próxima página.

<p align="center"><img src="https://github.com/user-attachments/assets/c9aa9874-66d4-4947-859f-cf4bfeaaef1a" width="300" height="500"></p>
